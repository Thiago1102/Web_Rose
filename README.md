# Rose Estetica

Sitio web estático para presentar los servicios de estética de Rose Estetica, facilitar la reserva de citas por WhatsApp y mostrar una presencia profesional en línea.

## Vista general

La web está pensada para ofrecer una experiencia visual elegante, clara y adaptada a móvil.

### Incluye
- presentación del negocio
- catálogo visual de servicios
- sección sobre la profesional
- ubicación con mapa
- formulario de reserva conectado a WhatsApp
- botón flotante de Instagram
- diseño responsive para móvil y escritorio

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Google Maps Embed

## Estructura del proyecto

- index.html → estructura principal de la web
- styles.css → estilos y responsive
- script.js → interacciones, navegación y envío por WhatsApp
- img/ → imágenes y recursos gráficos

## Cómo usarlo en local

1. Descarga o clona el repositorio.
2. Abre la carpeta en VS Code.
3. Ejecuta la web abriendo el archivo index.html en el navegador.

## Personalización rápida

Antes de publicar, revisa estos puntos:

- número de WhatsApp en el formulario
- enlace de Instagram
- textos de contacto
- ubicación exacta del negocio
- imágenes de servicios y portada

## Despliegue

Este proyecto se puede publicar fácilmente en:

- Netlify
- GitHub Pages
- Vercel

### Publicación recomendada en Netlify

1. Inicia sesión en Netlify.
2. Importa el repositorio desde GitHub.
3. Selecciona este proyecto.
4. Publica la web sin necesidad de proceso de build adicional.

## Objetivo del proyecto

Transmitir una imagen cuidada y profesional de la marca, facilitar el contacto con clientes y convertir visitas en reservas.
